# 基于Taichi的万有引力粒子模拟实验报告
## 一、实验目的
1.  掌握使用`uv`包管理器创建并管理Python虚拟环境的方法，实现项目环境隔离。
2.  遵循现代工程规范，搭建标准`src` 项目代码布局，实现代码分层与解耦。
3.  理解并运用Taichi框架实现GPU并行计算，完成基于万有引力定律的粒子群物理模拟。
4.  熟练使用Git进行版本控制，完成项目代码的仓库同步与文档编写。

## 二、实验原理
1.  **环境隔离原理**：利用`uv`工具创建独立的`.venv`虚拟环境，将项目依赖与全局Python环境隔离，避免版本冲突，保证项目环境的纯净性与可复现性。
2.  **工程化布局**：采用标准`src`源码布局，将项目拆分为配置（`config.py`）、物理逻辑（`physics.py`）、入口渲染（`main.py`），提升代码的可维护性与路径清晰度。
3.  **并行计算与物理模型**：通过Taichi的`@ti.kernel`装饰器定义并行内核，利用GPU加速大规模粒子间的引力计算；核心基于牛顿万有引力定律 $F = G \cdot \frac{m_1 \cdot m_2}{r^2 + \epsilon}$ 进行粒子运动状态的迭代更新，其中$\epsilon$ 为平滑因子，用于避免计算时的除零错误。

## 三、实验过程与核心代码
### 1. 环境搭建与项目初始化
- 安装`uv`并初始化项目，创建并激活虚拟环境。
- 按照规范创建`src/work0`目录结构，包含`__init__.py`、`config.py`、`physics.py`、`main.py`。
- 安装Taichi依赖：`uv add taichi`。

### 2. 核心代码实现
#### （1）配置文件 `src/work0/config.py`
```python
import taichi as ti

# 窗口与粒子基础配置
WINDOW_WIDTH, WINDOW_HEIGHT = 800, 600
WINDOW_TITLE = "Gravity Simulation (Taichi)"
PARTICLE_NUM = 1000

# 物理参数
GRAVITY_CONST = 6.67430e-11
SMOOTH_FACTOR = 1e-2
TIME_STEP = 0.01

# Taichi 设备
TI_DEVICE = ti.gpu
```
### 3. Git 版本控制
- 创建`.gitignore`文件，忽略`.venv`、`__pycache__`等文件。
- 执行`git init`初始化仓库，完成代码提交与远程仓库关联推送。
