# 二、实验原理
## 2.1 核心光线追踪流程
| 步骤 | 功能说明 |
|------|---------|
| 光线生成 | 从相机出发，为每个像素生成观察方向 |
| 物体求交 | 检测光线与球体、平面的交点 |
| 法线计算 | 计算交点表面法向量，用于光照计算 |
| 材质计算 | 根据材质类型执行反射/折射 |
| 着色输出 | 计算漫反射、阴影、环境光并输出颜色 |

## 2.2 斯涅尔定律（折射原理）
$$n_1 \sin\theta_1 = n_2 \sin\theta_2$$
- 光线从**空气（n≈1）**进入**玻璃（n≈1.5）**
- 当入射角超过临界角，发生**全反射**，不产生折射
- 玻璃材质需要同时处理**折射透射**与**全反射**

## 2.3 MSAA 抗锯齿原理
- 单个像素内**多次随机采样**
- 对采样结果**取平均值**
- 消除边缘阶梯状锯齿，使轮廓平滑自然

# 三、核心算法与代码块分析
## 3.1 折射算法（玻璃材质核心）
### 功能
实现斯涅尔定律，计算折射方向，并判断全反射是否发生。

```python
@ti.func
def refract(I, N, ni_over_nt):
    uv = normalize(I)
    dt = uv.dot(N)
    discriminant = 1.0 - ni_over_nt**2 * (1 - dt**2)
    # 全反射判断
    if discriminant > 0:
        refracted = ni_over_nt * (uv - N*dt) - N * ti.sqrt(discriminant)
        return True, refracted
    else:
        # 发生全反射
        return False, refracted
```

### 关键分析
1. **`discriminant`**：判断是否满足折射条件
2. **`discriminant < 0`**：触发**全反射**，不产生折射光线
3. **`ni_over_nt = 1/1.5`**：空气→玻璃的折射率比值
4. 该函数是**透明玻璃材质**的物理基础

## 3.2 玻璃材质逻辑
### 功能
根据入射方向选择**折射**或**全反射**，实现真实透明效果。

```python
if mat == GLASS:
    outward_n = n
    if rd.dot(n) > 0: outward_n = -n
    # 计算折射
    success, refracted = refract(rd, outward_n, 1.0/1.5)
    if success: rd = refracted
    else: rd = reflect(rd, n)
```

### 关键分析
1. 自动判断光线**从内部/外部入射**
2. 折射成功则沿**折射方向**传播
3. 折射失败则执行**全反射**
4. 最终形成**透明+边缘反光**的真实玻璃效果

## 3.3 MSAA 多采样抗锯齿算法
### 功能
对单个像素进行多次采样并平均，消除边缘锯齿。

```python
for _ in range(samples):
    ox = (ti.random() - 0.5) * 0.5
    oy = (ti.random() - 0.5) * 0.5
    u = (i + ox - WIDTH/2) / HEIGHT
    v = (j + oy - HEIGHT/2) / HEIGHT
    color += ray_color(ro, rd)

```

### 关键分析
1. **`samples=4`**：每个像素采样4次
2. **随机偏移**：让采样点覆盖像素不同区域
3. **颜色平均**：消除高频锯齿噪声
4. 画面轮廓更平滑，无明显像素阶梯
