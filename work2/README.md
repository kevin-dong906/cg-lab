贝塞尔曲线绘制实验报告
 
一、实验目标
 
- 理解贝塞尔曲线的形成原理，知道它是由控制点决定的平滑曲线
- 掌握并使用 De Casteljau 算法计算曲线上的点
- 了解屏幕像素绘制的基本方式，学会直接点亮像素
- 实现鼠标添加控制点、键盘清空画布的简单交互功能
 
二、实验原理
 
- 贝塞尔曲线依靠一组控制点生成，通过 0~1 之间的参数确定曲线上具体位置
- De Casteljau 算法通过多层比例插值，不断缩点，最终得到曲线上的点
- 先在 CPU 计算好所有曲线点，再批量发送给 GPU 绘制，避免频繁传输导致卡顿
- 将计算出的坐标映射到屏幕像素位置，修改颜色即可完成绘制
 
三、实验内容与实现
 
- 初始化画布与数据缓冲区，预先分配好存储像素和控制点的空间
- 实现 De Casteljau 算法，根据输入控制点计算不同参数下的曲线点坐标
- 编写并行绘制函数，在 GPU 上批量点亮曲线对应像素
- 搭建交互窗口，监听鼠标点击添加控制点，按下 C 键清空画布
- 实时显示控制点、辅助线和生成的贝塞尔曲线
 
四、实验结果
 
- 鼠标左键可在画布上添加控制点，实时显示红色圆点
- 两个及以上控制点会自动生成绿色平滑贝塞尔曲线
- 控制点之间显示灰色辅助线，整体效果清晰直观
- 按下 C 键可快速清空所有内容，支持重新绘制
- 程序运行流畅，无明显卡顿，交互响应及时
 
五、实验总结
 
- 本次实验成功实现了交互式贝塞尔曲线绘制，理解了曲线生成与像素渲染的基本流程
- 采用批量数据传输优化了性能，解决了画面卡顿问题
- 调整了视觉配色与点线样式，让整体效果更简洁、辨识度更高
- 对图形渲染中 CPU 与 GPU 的协作方式有了更直观的认识
